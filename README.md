# NewsHive

Telegram-бот-дайджест новостей, управляемый 5 LLM-агентами.

## Функции
- Автоматическая обработка новостей через pipeline из 5 агентов
- Персонализированные рекомендации
- Настройки пользователя
- Хранение данных в SQLite

## Зависимости
- Python 3.9+
- OpenRouter API ключ
- Telegram Bot Token

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте `.env`
3. Запустите: `python bot/main.py`
